Meet Destri

A native application designed for both iOS and Android versions, to help users build and discover new routes for their upcoming walks. This concept application was a course project with a project brief for a native app that solves a specific user problem.

I was the UI/UX Designer for the entire project. I came up with the User Flows, Wireframes, Mobile Gestures, Brand Guidelines, and conducted User Testing to make changes to my final UI Design.

I worked on the project from the beginning of June 2021 till mid-July 2021.

Research

First, I conduced research on current walking apps in the market, available for both iOS and Android. Top competitors included Strava, Pacer and MapMyWalk, which incorporated walking trackers and challenges for users. During my analysis, I noticed that users are not able to build new routes on the mobile applications and can only save routes already tracked and completed.

Time to solve this issue with Destri!

Ideation

Key features I wanted to focus on were Users ability to build routes based on distance, time, or steps desired. Additionally, the ability to save routes they have newly built of already completed, as well as, set reminders for future walks.

These key features also signified the Main Jobs To Be Done, which not only built out individual User Stories but also curated my User Flows.

Wireframe

After working through rounds of Crazy 8s and sketching Low-Fidelity Wireframes, I developed digitalized screens for both iOS & Android Platforms from Mid to High Fidelity.

Once the high-fidelity wireframes were completed, I was able to conceptualize the type of Haptic Feedback, Gestures and Sound the application would be able make, in order to help users navigate through each screen.

Prototype

By using InVision, I created a 2 Prototypes for my High-Fidelity Wireframe consistent for iOS and Android. I recruited 3 iOS Users and 2 Android users to test the initial prototypes.

Based on constructive feedback, I was able to identify weaknesses in the design and make necessary changes to multiple screens.

Branding & Design

I created the brand name & logo for the application. When walking towards a desired destination a form of walking can either be slow, moderate or a swift stride. By making an anagram from the word Stride, I came up with Destri.

The style for both iOS and Android versions follow iOS Developer and Material Design guidelines, respectively.

Certain users can consider exercise a daunting task. In order for users to be excited to walk, I wanted the colors of the application to be vivid, fun and energetic. I decided to first incorporate primary colors (blue, red and yellow) before adding other complementary secondary colors.

Conclusion

Based on feedback from participants during user testing, I was able to construct Final UI Design wireframes for both iOS and Android, that meet guidelines and solve users problem of building new routes before walking them.
